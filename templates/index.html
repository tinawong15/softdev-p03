What is pandas?

  • Python Data Analysis Library
  • Derived from "panel data"
  • Created by William McKinney in 2008 and has quickly become one of the most widely used tools for working with data
  • Open source
  • Used for data analysis and modeling, which is critical in the world of data science
  • It is often used alongside other python libraries such as numpy, which helps with calculations, and matplotlib, which can plot data points

What can pandas do?

  • Includes data structures to allow you to view data in an organized format, such as a DataFrame and a Series
  • Can read data from a variety of formats
    • To read from a csv and convert it to a DataFrame, use the method read_csv
  • Statistics
    • It has many functions that are helpful in statistics, such as the mean of all columns, max, min, median, and standard deviation of each column
  • Data manipulation
    • You can select values from a column easily, filter, sort, and group data
    • Other functions allow you to clean the data by taking away null values
    • There are commands for joining or combining data frames or rows/columns

What are advantages of using pandas?

  • Efficient representation of data that is easy to read and useful for data analysis
  • Simple syntax and flexible
  • Able to handle large datasets
  • Data filtering and subsetting

What are some disadvantages?

  • Not good for arrays that are more than 2 dimensional
  • Not good for more complex data modeling
  • There are many other tools used for data analysis such as Excel VBA

How do you get started?

  • Installed through the terminal by executing the command:
    • pip install pandas
  • Use in a Python file by importing the pandas library
    • import pandas as pd
  • One useful tool you can use to clearly and cleanly visualize your data and pandas at work is Jupyter Notebook. pandas does work with regular python files, however, and will display in the terminal.

Example Code

Create a Series

Using an example one-dimensional array [1, 2, 3, 4] and the syntax below, pandas can create a series:
your_series = pd.Series([1,2,3,4])
print(your_series)

Create a DataFrame

Using the syntax below, pandas can create a DataFrame with four columns and four rows consisting of different types:
df = pd.DataFrame({
"Integers": [1, 23, 456, 7890],
"Strings": ['hello', 'pandas', 'world', '!'],
"Floats": [1.23, 23.5, 45.6, 7.890],
"Booleans": [True, False, True, False]
})
print(df)

Create a DataFrame with Existing list

list_df = pd.DataFrame([your_list], columns=['Integers', 'Strings', 'Floats', 'Booleans'])

Once you create a DataFrame, the DataFrame is still mutable, so you can append one DataFrame to another.

Create a DataFrame with Existing csv

Using any csv file you have, you can convert it to a DataFrame. .head(n) will show the first n rows of the DataFrame. If .head() gets no arguments, the default rows shown will be 5.
csv_path = os.path.join('uscities.csv')
csv_df = pd.read_csv(csv_path, delimiter=",")
print(csv_df.head(10))

{{ head | safe}}

.shape shows the number of rows and the number of columns the DataFrame has.
print(csv_df.shape)

.tail() works like .head() does, except that it shows the last n rows.
tail = csv_df.tail()
print(tail)

.sample(n) will show you n random rows from the DataFrame.
